Opening the Hood

Agent Anatomy

Now that we know what AcmeCorp's agent needs externally, let's open the hood and see what's inside. Every Data Agent is built from six internal components working together.

Internal Components

The Six Building Blocks

1 LLM (Core Reasoning Engine)

The brain of the agent. Makes decisions, generates plans, and interprets natural language into structured actions.

AcmeCorp Example
Interprets "top products" as needing a GROUP BY + ORDER BY query against the sales table.
2 Reasoning Graph

Orchestrates multi-step logic via LangGraph. Defines the order of operations and branching decisions the agent follows.

AcmeCorp Example
Decides: first discover schema, then write SQL, then format the results for the user.
3 Memory

Short-term (conversation context) + long-term (persistent storage). Enables continuity across interactions and sessions.

AcmeCorp Example
Remembers you asked about Q2 earlier, so "same thing for Q3" makes sense without re-explaining.
4 Tools

Actions the agent can execute: database queries, API calls, file operations, and computations in the real world.

AcmeCorp Example
The actual query_execute, schema_discovery functions that hit the warehouse.
5 Policies

Rules and guardrails: rate limits, validation logic, access controls, and safety constraints that bound behavior.

AcmeCorp Example
Store-level analysts can't run queries touching salary tables. Only HQ admins have that access.
6 Context

Background knowledge: schema documentation, past resolutions, RAG-indexed domain knowledge that informs decisions.

AcmeCorp Example
Knows that "revenue" in AcmeCorp means sales.total_amount, not orders.subtotal.
Think of it like a sports team...

LLM is the quarterback — reads the field and makes the play call.

Reasoning Graph is the playbook — defines what sequence of plays to run.

Memory is the team's game tape — remembers what worked before and what the opponent did last time.

Tools are the players' skills — passing, blocking, running. The actual capabilities.

Policies are the referee — enforces the rules, flags illegal moves.

Context is the scouting report — background intelligence about the field, the opponent, the conditions.
