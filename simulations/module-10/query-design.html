Interactive Simulation

Query Design & Agent UX

AcmeCorp's Data Agent is technically sound — but store managers find it confusing, produce inconsistent results, and don't trust its answers. You must redesign the semantic layer and conversation UX to make it genuinely useful.

Mission Brief
Five UX and semantic layer design decisions: how to handle ambiguous queries, when to clarify vs. assume, how to surface data quality issues, how to design the semantic layer for business users, and how to handle multi-turn follow-up questions. Aim for 5/5.
