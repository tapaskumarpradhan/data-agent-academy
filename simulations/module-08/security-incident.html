Security Incident Response

AcmeCorp's Data Agent is under attack. Threat actors are probing for prompt injection vulnerabilities, attempting to exfiltrate PII, and trying to bypass access controls. Navigate 5 security scenarios with the right defensive response.

Mission Brief
AcmeCorp's Data Agent handles PII (customer emails, employee salaries) and serves 200 stores. A single security failure could violate GDPR, expose salary data, or allow unauthorized SQL execution. Make the right calls across prompt injection, PII handling, access control, compliance, and incident response — aim for 5/5.
